# Fasih-TTS-V1 — live demo

Type Arabic (even undiacritized — CATT adds tashkīl automatically) and hear a professional male
**Modern Standard Arabic (Fusha)** voice. Fine-tuned from Coqui XTTS v2; #1 for intelligibility on
the SILMA open-source Arabic TTS benchmark.

- Model: https://huggingface.co/NightPrince/Fasih-TTS-V1
- Benchmark dataset: https://huggingface.co/datasets/NightPrince/Fasih-TTS-Benchmark
- Code: https://github.com/NightPrinceY/Fasih-TTS-V1

Non-commercial (CPML). Qur'anic recitation is out of scope.
